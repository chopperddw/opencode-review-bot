⚡ Headless 🔌 Provider-Agnostic 🔒 Privacy-First 📊 HTML Reports

🎯 What Is This?

A fully automated code review bot that clones a git repo, diffs a branch, sends the changes to an AI model, and generates a professional HTML report — all in one command.

The key idea: it runs entirely headless via OpenCode's run command, making it perfect for CI/CD pipelines, GitHub webhooks, Slack bots, or any automation you can imagine. No interactive terminal required.

💡 Why OpenCode, Not Claude Code?

This demo exists to prove that OpenCode is a programmable coding platform, while Claude Code is an interactive CLI. The difference matters when you want automation.

Capability Claude Code OpenCode
Headless API for automation Limited (-p only) Full SDK + HTTP server
Provider flexibility Anthropic only Any (Claude, GPT, Gemini, local)
Use subscription programmatically Interactive CLI only BYO API keys, full programmatic
Data storage Vendor server Zero storage — privacy first
Open source Proprietary 160K GitHub stars
Cost control by task Fixed model Route to cheapest model per task

The Subscription Problem (Specifically)

Claude Code subscriptions (Pro/Max at $20-200/mo) are tied to interactive CLI use. You cannot use a Claude subscription to run this bot — you need a separate Anthropic API key (pay-per-token).

With OpenCode, you bring whatever API key your team already has (OpenRouter, OpenAI, Google, etc.) and it works everywhere: interactive TUI, headless CLI, HTTP server, SDK.

🚀 Try It Yourself

1. Install & Run

git clone https://github.com/chopperddw/opencode-review-bot
cd opencode-review-bot
uv init && uv add jinja2

2. Get an API key (any provider)

export OPENROUTER_API_KEY=*** your-key

3. Review any repo

python review_bot.py \
  --repo https://github.com/user/repo.git \
  --branch feature/x \
  --base main \
  --output review.html

# Or with a different model:
python review_bot.py --repo ... --model openrouter/openai/gpt-5.5
python review_bot.py --repo ... --model openrouter/google/gemini-2.5-pro

4. Open the report

xdg-open review.html  # Linux
open review.html      # macOS

🔌 Real-World Integrations
